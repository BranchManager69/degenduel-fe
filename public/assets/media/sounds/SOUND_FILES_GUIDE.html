Elite Token List Sound Files Guide

This document provides information on the sound files needed for the Elite Token List component in DegenDuel.

Required Sound Files

The following sound files should be placed in this directory:

token-up.mp3

A short, positive sound played when a token's price increases.

Suggested characteristics:

  • Short duration (200-300ms)
  • Ascending tone
  • Bright, optimistic sound
  • Low volume to avoid being disruptive

token-down.mp3

A short, negative sound played when a token's price decreases.

Suggested characteristics:

  • Short duration (200-300ms)
  • Descending tone
  • Slightly muted or low pitch
  • Low volume to avoid being disruptive

rank-change.mp3

A distinctive sound played when a token changes its position in the ranking.

Suggested characteristics:

  • Short to medium duration (300-500ms)
  • Distinctive "swoosh" or "shift" sound
  • Different character from the up/down sounds
  • Should convey movement or rearrangement

Implementation Notes

The sounds are played using the HTML5 Audio API with volume levels adjusted to be non-intrusive:

// Sound setup
upSoundRef.current = new Audio('/assets/media/sounds/token-up.mp3');
downSoundRef.current = new Audio('/assets/media/sounds/token-down.mp3');
rankChangeRef.current = new Audio('/assets/media/sounds/rank-change.mp3');

// Volume settings
if (upSoundRef.current) upSoundRef.current.volume = 0.3;
if (downSoundRef.current) downSoundRef.current.volume = 0.3;
if (rankChangeRef.current) rankChangeRef.current.volume = 0.4;

Users can toggle sounds on/off using the sound control button in the Elite Token List header.

Adding Your Own Sounds

To add your own sounds:

  1. Create or acquire the sound files (MP3 format recommended)
  2. Name them exactly as specified above
  3. Place them in this directory
  4. Keep file sizes small (under 50KB each) for optimal performance

Note: Sound files should be subtle and professional, aligned with the premium, elite character of the token list.
